# KDS-SunucuTabanli
"EkranGoruntuleri" adlı klasörde siteye ait fotoğraflar mevcuttur.
Diğer klasörler(css,data,images,js,legend) QGIS2Web Tool 'unun haritaların webte görüntülenmesi için otomatik oluşturduğu klasörlerdir.
Siteye ait tüm kodlar doğrudan başlangıçtadır.(Herhangi bir klasörün içinde değildir.)
Site yapımında Bootstrap, Chart.js, Wow.js, LightBox, Font Awesome, Google Fonts ve haritalar için QGIS2Web araçları kullanılmıştır.
Site için kullanılan veritabanı, "veritabani" adlı klasörde sigorta.sql şeklinde bulunmaktadır.
